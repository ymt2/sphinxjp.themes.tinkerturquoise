Feature
=======
Two column blogging theme tool for Tinkerer, based on turquoise color.


Installation
============
Make environment with easy_install::

   $ easy_install sphinxjp.themes.tinkerturquoise


Make environment with pip::

   $ pip install sphinxjp.themes.tinkerturquoise


setup conf.py with::

   html_theme = 'tinkerturquoise'
   extensions = [..., 'sphinxjp.themecore']


and run::

   $ tinker -b


Requirement
===========
Libraries:

* Safari on iOS, Mac OS
* Android Browser
* Chrome
* Firefox 10 or later

Browsers:

* Safari on iOS, Mac OS
* Android Browser
* Chrome
* Firefox 10 or later
